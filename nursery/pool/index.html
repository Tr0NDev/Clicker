Fusion rules

  • Five fish are mandatory before the fusion can start.
  • The strongest fish by base size + weight survives as the output.
  • The four other fish are removed from the inventory.
  • The output inherits mutations from one random donor fish in the fusion pool.
  • If the donor has multiple mutations, the output inherits all of them.
  • There is also a 10% chance the output inherits a second mutation, if one is available.
